## Q1
**difficulty:** Easy
**type:** Problem Solving
**topic:** Mean

The average (arithmetic mean) of five numbers is 18. If one of the numbers is removed, the average of the remaining four numbers is 15. What is the number that was removed?

- A) 20
- B) 24
- C) 28
- D) 30
- E) 33

**answer:** D
**explanation:** Sum of all five numbers = 5 x 18 = 90. Sum of remaining four numbers = 4 x 15 = 60. The removed number = 90 - 60 = 30.

## Q2
**difficulty:** Easy
**type:** Problem Solving
**topic:** Probability

A bag contains 4 red balls, 3 blue balls, and 5 green balls. If one ball is drawn at random, what is the probability that it is NOT green?

- A) 5/12
- B) 7/12
- C) 1/3
- D) 2/3
- E) 3/4

**answer:** B
**explanation:** Total balls = 4 + 3 + 5 = 12. Non-green balls = 4 + 3 = 7. Probability = 7/12. Alternatively, P(not green) = 1 - P(green) = 1 - 5/12 = 7/12.

## Q4
**difficulty:** Medium
**type:** Problem Solving
**topic:** Counting

A committee of 3 people is to be selected from a group of 7 candidates. How many different committees are possible?

- A) 21
- B) 35
- C) 42
- D) 120
- E) 210

**answer:** B
**explanation:** The number of ways to choose 3 from 7 is C(7,3) = 7!/(3! x 4!) = (7 x 6 x 5)/(3 x 2 x 1) = 210/6 = 35. Trap answer E (210) is 7 x 6 x 5 without dividing by 3!, which counts ordered arrangements.

## Q6
**difficulty:** Hard
**type:** Problem Solving
**topic:** Standard Deviation

Set A = {10, 20, 30, 40, 50} and Set B = {20, 25, 30, 35, 40}. Which of the following statements is true?

- A) Set A and Set B have the same mean and the same standard deviation
- B) Set A and Set B have the same mean, but Set A has a greater standard deviation
- C) Set A and Set B have the same mean, but Set B has a greater standard deviation
- D) Set A has a greater mean and a greater standard deviation
- E) Set B has a greater mean and a smaller standard deviation

**answer:** B
**explanation:** Mean of Set A = (10+20+30+40+50)/5 = 150/5 = 30. Mean of Set B = (20+25+30+35+40)/5 = 150/5 = 30. The means are equal. Set A values deviate from the mean by -20, -10, 0, 10, 20 (range of 40). Set B values deviate from the mean by -10, -5, 0, 5, 10 (range of 20). Since Set A's values are more spread out from the mean, Set A has a greater standard deviation. This eliminates all choices except B.

## Q7
**difficulty:** Hard
**type:** Problem Solving
**topic:** Counting

How many 3-digit positive integers have all distinct digits and are divisible by 5?

- A) 112
- B) 128
- C) 136
- D) 144
- E) 152

**answer:** C
**explanation:** A 3-digit number divisible by 5 must end in 0 or 5. Case 1: Units digit is 0. The hundreds digit can be any of 1-9 (9 choices). The tens digit can be any digit except the hundreds digit and 0 (8 choices). Total = 9 x 8 = 72. Case 2: Units digit is 5. The hundreds digit can be any of 1-9 except 5 (8 choices). The tens digit can be any of 0-9 except the hundreds digit and 5 (8 choices). Total = 8 x 8 = 64. Grand total = 72 + 64 = 136.

## Q8
**difficulty:** Hard
**type:** Problem Solving
**topic:** Probability

A box contains 5 red chips and 3 blue chips. Two chips are drawn at random without replacement. What is the probability that at least one chip is red?

- A) 5/8
- B) 25/28
- C) 13/14
- D) 15/28
- E) 27/28

**answer:** B
**explanation:** Use the complement: P(at least one red) = 1 - P(no red) = 1 - P(both blue). P(both blue) = (3/8) x (2/7) = 6/56 = 3/28. Therefore P(at least one red) = 1 - 3/28 = 25/28. Trap answer A (5/8) is the probability of drawing one red chip on a single draw.

## Q9
**difficulty:** Easy
**type:** Problem Solving
**topic:** Mean / Median

The set {4, 7, 9, 12, x} has a mean of 9. What is the value of x?

- A) 9
- B) 11
- C) 12
- D) 13
- E) 15

**answer:** D
**explanation:** The mean equals the sum divided by the count. If the mean is 9 and there are 5 numbers, the sum must be 5 x 9 = 45. The four known values sum to 4 + 7 + 9 + 12 = 32, so x = 45 - 32 = 13. A reliable technique: total = mean x count, then solve for the missing term.

## Q10
**difficulty:** Easy
**type:** Problem Solving
**topic:** Basic Probability

A jar contains 6 red marbles, 4 blue marbles, and 2 yellow marbles. If one marble is drawn at random, what is the probability that it is either red or yellow?

- A) 1/3
- B) 1/2
- C) 7/12
- D) 2/3
- E) 3/4

**answer:** D
**explanation:** For mutually exclusive events, P(A or B) = P(A) + P(B). Total marbles = 12. P(red) = 6/12 and P(yellow) = 2/12, so P(red or yellow) = 8/12 = 2/3. Equivalently, count favorable outcomes (6 + 2 = 8) over total (12).

## Q11
**difficulty:** Easy
**type:** Problem Solving
**topic:** Counting Principle

A restaurant offers 4 appetizers, 6 entrees, and 3 desserts. How many different three-course meals (one of each) can a customer order?

- A) 13
- B) 24
- C) 36
- D) 54
- E) 72

**answer:** E
**explanation:** The fundamental counting principle says that if one choice has m options and an independent choice has n options, there are m x n combined outcomes. Here: 4 x 6 x 3 = 72. Trap answer A (13) is the sum of the choices, which ignores independence.

## Q12
**difficulty:** Medium
**type:** Problem Solving
**topic:** Mean / Median

The mean of five distinct positive integers is 20 and their median is 18. If the largest of the five integers is 40, what is the greatest possible value of the smallest integer?

- A) 1
- B) 5
- C) 10
- D) 12
- E) 17

**answer:** C
**explanation:** The five integers in order are a < b < 18 < d < 40, summing to 100. So a + b + d = 100 - 18 - 40 = 42. To maximize a, minimize d: the smallest d > 18 (distinct integers) is 19. Then a + b = 42 - 19 = 23, with a < b < 18. Maximize a by taking b = 18 - 1 = wait, b < 18, so largest b = 17, giving a = 23 - 17 = 6. But check: take d = 19 and b = 13, a = 10 works if b > a. Actually maximize a: need a < b < 18 and a + b = 23. If a = 10, b = 13 (works). If a = 11, b = 12 (works, a < b < 18). If a = 12, b = 11 violates a < b. So greatest a = 11... reconsider: try d = 20, then a + b = 22, a = 10, b = 12 works; a = 11, b = 11 not distinct. Maximum a is 10 (with d = 19, b = 13 OR d = 20, b = 12). Answer: 10.

## Q13
**difficulty:** Medium
**type:** Problem Solving
**topic:** Combinations

From a group of 5 men and 4 women, a committee of 4 people is to be formed consisting of exactly 2 men and 2 women. How many different committees are possible?

- A) 20
- B) 36
- C) 60
- D) 126
- E) 210

**answer:** C
**explanation:** Choose 2 men from 5: C(5,2) = 10. Choose 2 women from 4: C(4,2) = 6. By the counting principle, multiply independent selections: 10 x 6 = 60. Trap D (126) is C(9,4), which ignores the gender constraint.

## Q14
**difficulty:** Medium
**type:** Problem Solving
**topic:** Mean / Median

A list of 10 numbers has a mean of 24. When an 11th number is added, the new mean becomes 25. What is the value of the 11th number?

- A) 25
- B) 26
- C) 30
- D) 35
- E) 36

**answer:** D
**explanation:** Original sum = 10 x 24 = 240. New sum = 11 x 25 = 275. The added number = 275 - 240 = 35. Shortcut: when n numbers with mean m gain a new value that raises the mean by d, the new value equals the new mean plus n x d = 25 + 10 x 1 = 35.

## Q15
**difficulty:** Medium
**type:** Problem Solving
**topic:** Dependent Events

A drawer contains 7 black socks and 5 white socks. If two socks are drawn at random without replacement, what is the probability that both are black?

- A) 7/24
- B) 7/22
- C) 1/2
- D) 35/132
- E) 49/144

**answer:** B
**explanation:** For events without replacement, the second probability depends on the first outcome. P(first black) = 7/12. After removing one black sock, 6 black remain of 11 total, so P(second black | first black) = 6/11. Multiply: (7/12) x (6/11) = 42/132 = 7/22. Trap E (49/144) is (7/12)^2, which incorrectly assumes replacement.

## Q16
**difficulty:** Medium
**type:** Problem Solving
**topic:** Standard Deviation

Set X = {8, 10, 12, 14, 16} and Set Y is formed by adding 5 to each element of Set X. Which of the following statements is true about the two sets?

- A) Set Y has a greater mean and greater standard deviation
- B) Set Y has a greater mean and smaller standard deviation
- C) Set Y has a greater mean and the same standard deviation
- D) The two sets have the same mean and the same standard deviation
- E) The two sets have the same mean but different standard deviations

**answer:** C
**explanation:** Adding a constant to every element of a set shifts the mean by that constant but does not change the spread. Set X mean = 12, Set Y mean = 17. The distances between values are unchanged, so standard deviation is identical. Key rule: additive shifts move center, not spread; multiplicative scaling changes both.

## Q17
**difficulty:** Medium
**type:** Data Sufficiency
**topic:** Mean / Median

In a set of 5 positive integers, is the median equal to the mean?

(1) The 5 integers form an arithmetic sequence.
(2) The smallest integer is 4 and the largest is 20.

- A) Statement (1) ALONE is sufficient, but statement (2) alone is not sufficient.
- B) Statement (2) ALONE is sufficient, but statement (1) alone is not sufficient.
- C) BOTH statements TOGETHER are sufficient, but NEITHER statement ALONE is sufficient.
- D) EACH statement ALONE is sufficient.
- E) Statements (1) and (2) TOGETHER are not sufficient.

**answer:** A
**explanation:** In any arithmetic sequence with an odd number of terms, the middle term equals both the median and the mean (the sequence is symmetric about its center). So (1) alone answers "yes" definitively. (2) alone is insufficient: the set {4, 5, 6, 7, 20} has median 6 and mean 8.4 (no), while {4, 8, 12, 16, 20} has median 12 and mean 12 (yes). Answer: A.

## Q18
**difficulty:** Hard
**type:** Problem Solving
**topic:** Basic Probability

A fair coin is flipped 4 times. What is the probability that it lands on heads exactly 3 times?

- A) 1/8
- B) 3/16
- C) 1/4
- D) 5/16
- E) 3/8

**answer:** C
**explanation:** Count favorable sequences: choose which 3 of 4 flips are heads — C(4,3) = 4 ways. Each specific sequence has probability (1/2)^4 = 1/16. Total = 4 x 1/16 = 4/16 = 1/4. For "exactly k heads in n flips," the formula is C(n,k) / 2^n when the coin is fair.

## Q19
**difficulty:** Hard
**type:** Data Sufficiency
**topic:** Combinations

A committee of 3 is to be chosen from a group of n people. How many different committees are possible?

(1) If the group had 1 additional person, the number of possible committees would increase by 21.
(2) n is greater than 5.

- A) Statement (1) ALONE is sufficient, but statement (2) alone is not sufficient.
- B) Statement (2) ALONE is sufficient, but statement (1) alone is not sufficient.
- C) BOTH statements TOGETHER are sufficient, but NEITHER statement ALONE is sufficient.
- D) EACH statement ALONE is sufficient.
- E) Statements (1) and (2) TOGETHER are not sufficient.

**answer:** A
**explanation:** From (1): C(n+1, 3) - C(n, 3) = 21. Using the identity C(n+1,3) - C(n,3) = C(n,2), we get C(n,2) = 21, so n(n-1)/2 = 21, giving n = 7. That uniquely determines C(7,3) = 35. Sufficient. Statement (2) alone only says n > 5, so n could be 6, 7, 8, etc., yielding different committee counts. Answer: A.

## Q20
**difficulty:** Hard
**type:** Problem Solving
**topic:** Combinations

A student must answer 5 out of 7 questions on an exam, but must answer at least 2 of the first 3 questions. How many different sets of 5 questions can the student choose?

- A) 18
- B) 20
- C) 21
- D) 24
- E) 28

**answer:** A
**explanation:** Split by how many of the first 3 are chosen. Exactly 2 of the first 3 plus 3 of the remaining 4: C(3,2) x C(4,3) = 3 x 4 = 12. All 3 of the first 3 plus 2 of the remaining 4: C(3,3) x C(4,2) = 1 x 6 = 6. Total = 12 + 6 = 18. Complement check: C(7,5) = 21 total, minus cases with only 1 of the first 3 (which forces taking all 4 of the last 4): C(3,1) x C(4,4) = 3. So 21 - 3 = 18. Answer A.
