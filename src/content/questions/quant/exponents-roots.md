## Q1
**difficulty:** Easy
**type:** Problem Solving
**topic:** Exponent Rules

What is the value of 2^5 * 2^3?

- A) 2^8
- B) 2^15
- C) 4^8
- D) 4^15
- E) 2^2

**answer:** A
**explanation:** When you multiply powers with the same base, add the exponents: 2^5 * 2^3 = 2^(5+3) = 2^8. A common trap is to multiply the bases as well — but the base stays the same. Only the exponents combine.

## Q2
**difficulty:** Easy
**type:** Problem Solving
**topic:** Roots & Radicals

Simplify sqrt(50).

- A) 5
- B) 5*sqrt(2)
- C) 2*sqrt(5)
- D) 10*sqrt(5)
- E) 25*sqrt(2)

**answer:** B
**explanation:** To simplify a square root, factor out the largest perfect square. 50 = 25 * 2, so sqrt(50) = sqrt(25)*sqrt(2) = 5*sqrt(2). Always scan for perfect square factors (4, 9, 16, 25, 36, 49) before doing anything else.

## Q3
**difficulty:** Easy
**type:** Problem Solving
**topic:** Scientific Notation

Express 0.00045 in scientific notation.

- A) 4.5 * 10^3
- B) 4.5 * 10^-3
- C) 4.5 * 10^-4
- D) 45 * 10^-5
- E) 0.45 * 10^-3

**answer:** C
**explanation:** Scientific notation requires the coefficient to be between 1 and 10. Move the decimal 4 places to the right to get 4.5, so the exponent is -4 (negative because the original number is less than 1). The rule: count the places you moved and flip the sign for small numbers.

## Q4
**difficulty:** Easy
**type:** Problem Solving
**topic:** Equations with Exponents

If 2^x = 32, what is x?

- A) 3
- B) 4
- C) 5
- D) 6
- E) 16

**answer:** C
**explanation:** Rewrite 32 as a power of 2: 32 = 2^5. Once both sides have the same base, the exponents must be equal, so x = 5. Memorize the first few powers of 2 (2, 4, 8, 16, 32, 64, 128, 256) — they appear constantly on the GMAT.

## Q5
**difficulty:** Easy
**type:** Problem Solving
**topic:** Exponent Rules

What is the value of 3^-2?

- A) -9
- B) -6
- C) -1/9
- D) 1/6
- E) 1/9

**answer:** E
**explanation:** A negative exponent means take the reciprocal: 3^-2 = 1/(3^2) = 1/9. The negative sign flips the base into the denominator — it never makes the result negative. This is one of the most-tested traps on exponent questions.

## Q6
**difficulty:** Medium
**type:** Problem Solving
**topic:** Exponent Rules

Which of the following equals (x^4 * y^-2) / (x^-1 * y^3)?

- A) x^3 * y^-5
- B) x^5 * y^-5
- C) x^5 * y^-1
- D) x^3 * y^-1
- E) x^5 * y

**answer:** B
**explanation:** Subtract exponents when dividing same bases: x^(4-(-1)) = x^5 and y^(-2-3) = y^-5. The key move is handling the double-negative in the x-exponent carefully — subtracting a negative adds. Don't flip signs mid-step; just apply a^m / a^n = a^(m-n) directly.

## Q7
**difficulty:** Medium
**type:** Problem Solving
**topic:** Roots & Radicals

Simplify sqrt(12) + sqrt(27).

- A) sqrt(39)
- B) 3*sqrt(13)
- C) 5*sqrt(3)
- D) 6*sqrt(3)
- E) 13*sqrt(3)

**answer:** C
**explanation:** You can only add radicals if they share the same radicand, so simplify each first: sqrt(12) = 2*sqrt(3) and sqrt(27) = 3*sqrt(3). Now they combine: 2*sqrt(3) + 3*sqrt(3) = 5*sqrt(3). Treat matching radicals like variables — add the coefficients, keep the radical.

## Q8
**difficulty:** Medium
**type:** Problem Solving
**topic:** Fractional Exponents

What is the value of 27^(2/3)?

- A) 3
- B) 6
- C) 9
- D) 18
- E) 81

**answer:** C
**explanation:** A fractional exponent m/n means take the nth root, then raise to the mth power: 27^(2/3) = (27^(1/3))^2 = 3^2 = 9. Always take the root first — the numbers stay small and manageable. Raising 27 to the 2nd power first (729) would force an ugly cube root.

## Q9
**difficulty:** Medium
**type:** Problem Solving
**topic:** Scientific Notation

If (3 * 10^4) * (2 * 10^-7) = k, what is k?

- A) 6 * 10^-28
- B) 6 * 10^-11
- C) 6 * 10^-3
- D) 5 * 10^-3
- E) 6 * 10^11

**answer:** C
**explanation:** Multiply the coefficients and add the exponents separately: 3 * 2 = 6 and 10^4 * 10^-7 = 10^(4-7) = 10^-3. So k = 6 * 10^-3. Splitting scientific notation into coefficient and power-of-10 pieces keeps the arithmetic trivial.

## Q10
**difficulty:** Medium
**type:** Problem Solving
**topic:** Exponent Estimation

Approximately what is the value of 2^10 / 10^3?

- A) 0.01
- B) 0.1
- C) Slightly less than 1
- D) Slightly greater than 1
- E) 10

**answer:** D
**explanation:** 2^10 = 1,024 and 10^3 = 1,000, so the ratio is 1.024 — slightly greater than 1. The benchmark 2^10 ≈ 10^3 is one of the most useful GMAT shortcuts, but notice the direction: 2^10 is a touch LARGER than 1,000, not smaller. That 2.4% gap compounds quickly at higher powers.

## Q11
**difficulty:** Medium
**type:** Problem Solving
**topic:** Exponent Rules

If (2^a)^3 = 2^12, what is a?

- A) 2
- B) 3
- C) 4
- D) 6
- E) 9

**answer:** C
**explanation:** Power of a power means multiply the exponents: (2^a)^3 = 2^(3a). Setting 2^(3a) = 2^12 gives 3a = 12, so a = 4. Remember (x^m)^n = x^(mn) — a frequently confused rule, especially against x^m * x^n = x^(m+n).

## Q12
**difficulty:** Hard
**type:** Problem Solving
**topic:** Roots & Radicals

Rationalize the denominator: 6 / (sqrt(3) + 1).

- A) 6*sqrt(3) - 6
- B) 3*sqrt(3) - 3
- C) 3*sqrt(3) + 3
- D) 2*sqrt(3) + 2
- E) sqrt(3) - 1

**answer:** B
**explanation:** To clear a sum with a radical in the denominator, multiply by the conjugate: (sqrt(3) - 1) / (sqrt(3) - 1). The denominator becomes (sqrt(3))^2 - 1^2 = 3 - 1 = 2, using the difference of squares. The numerator is 6(sqrt(3) - 1), so the result is 6(sqrt(3) - 1)/2 = 3*sqrt(3) - 3.

## Q13
**difficulty:** Hard
**type:** Problem Solving
**topic:** Fractional Exponents

If x > 0 and x^(1/2) * x^(1/3) = x^k, what is k?

- A) 1/6
- B) 2/6
- C) 1/5
- D) 5/6
- E) 1

**answer:** D
**explanation:** Multiplying same-base powers means adding exponents, even when they're fractions: 1/2 + 1/3 = 3/6 + 2/6 = 5/6. So k = 5/6. The rule x^a * x^b = x^(a+b) holds for any real exponents — fractional, negative, or irrational.

## Q14
**difficulty:** Hard
**type:** Data Sufficiency
**topic:** Exponent Estimation

If n is a positive integer, is 2^n greater than 10^6?

(1) n > 19
(2) n is a multiple of 10

- A) Statement (1) ALONE is sufficient, but statement (2) alone is not sufficient.
- B) Statement (2) ALONE is sufficient, but statement (1) alone is not sufficient.
- C) BOTH statements TOGETHER are sufficient, but NEITHER alone is sufficient.
- D) EACH statement ALONE is sufficient.
- E) Statements (1) and (2) TOGETHER are NOT sufficient.

**answer:** A
**explanation:** Use the benchmark 2^10 ≈ 10^3, so 2^20 ≈ 10^6 (in fact 2^20 = 1,048,576, slightly above 10^6). For (1), n > 19 means n ≥ 20 (n is a positive integer), so 2^n ≥ 2^20 > 10^6 — sufficient. For (2), n could be 10 (2^10 = 1,024, far below 10^6) or 20 (above), so not sufficient. The 2^10 ≈ 10^3 approximation is load-bearing for this whole question type.

## Q15
**difficulty:** Hard
**type:** Data Sufficiency
**topic:** Equations with Exponents

What is the value of x?

(1) 2^x * 2^(x+1) = 32
(2) x is a positive integer

- A) Statement (1) ALONE is sufficient, but statement (2) alone is not sufficient.
- B) Statement (2) ALONE is sufficient, but statement (1) alone is not sufficient.
- C) BOTH statements TOGETHER are sufficient, but NEITHER alone is sufficient.
- D) EACH statement ALONE is sufficient.
- E) Statements (1) and (2) TOGETHER are NOT sufficient.

**answer:** A
**explanation:** From (1), combine exponents: 2^x * 2^(x+1) = 2^(2x+1) = 32 = 2^5, so 2x + 1 = 5 and x = 2 — sufficient. Statement (2) alone gives infinite possibilities. The key DS insight: when you can force both sides to the same base, exponents must match, giving a unique linear equation.
