## Q1
**difficulty:** Easy
**type:** Problem Solving
**topic:** Linear Equations

If 3x + 7 = 22, what is the value of 6x + 5?

- A) 25
- B) 30
- C) 35
- D) 40
- E) 45

**answer:** C
**explanation:** From 3x + 7 = 22, we get 3x = 15, so x = 5. Then 6x + 5 = 6(5) + 5 = 30 + 5 = 35. Alternatively, 6x + 5 = 2(3x) + 5 = 2(15) + 5 = 35.

## Q2
**difficulty:** Easy
**type:** Problem Solving
**topic:** Exponents

Which of the following is equivalent to (x^3)^4 / x^5?

- A) x^2
- B) x^7
- C) x^12
- D) x^15
- E) x^17

**answer:** B
**explanation:** (x^3)^4 = x^12. Then x^12 / x^5 = x^(12-5) = x^7.

## Q5
**difficulty:** Medium
**type:** Problem Solving
**topic:** Quadratics

If x^2 - 5x - 14 = 0, what is the product of the two solutions?

- A) -14
- B) -7
- C) 5
- D) 7
- E) 14

**answer:** A
**explanation:** By Vieta's formulas, for x^2 - 5x - 14 = 0, the product of the roots equals c/a = -14/1 = -14. To verify by factoring: x^2 - 5x - 14 = (x - 7)(x + 2) = 0, so x = 7 or x = -2, and 7 x (-2) = -14.

If f(x) = 2x^2 - 3x + 1, what is the value of f(3) - f(1)?

- A) 8
- B) 10

- D) 14
- E) 16

**answer:** B
**explanation:** f(3) = 2(9) - 3(3) + 1 = 18 - 9 + 1 = 10. f(1) = 2(1) - 3(1) + 1 = 2 - 3 + 1 = 0. Therefore f(3) - f(1) = 10 - 0 = 10.

## Q7
**difficulty:** Medium
**type:** Problem Solving
**topic:** Systems of Equations

At a bakery, 2 muffins and 3 scones cost $21, while 4 muffins and 1 scone cost $17. What is the cost of one scone?

- A) $1
- B) $2
- C) $3
- D) $4
- E) $5

**answer:** E
**explanation:** Let m = price of a muffin and s = price of a scone. Equation 1: 2m + 3s = 21. Equation 2: 4m + s = 17. From Equation 2: s = 17 - 4m. Substitute into Equation 1: 2m + 3(17 - 4m) = 21, so 2m + 51 - 12m = 21, giving -10m = -30, so m = 3. Then s = 17 - 4(3) = 17 - 12 = 5. One scone costs $5.

## Q8
**difficulty:** Medium
**type:** Problem Solving
**topic:** Inequalities

If 2 < |x - 3| < 7, how many integer values of x satisfy the inequality?

- A) 6
- B) 7
- C) 8
- D) 9
- E) 10

**answer:** C
**explanation:** |x - 3| > 2 means x > 5 or x < 1. |x - 3| < 7 means -4 < x < 10. Combining: -4 < x < 1 or 5 < x < 10. Integer values in -4 < x < 1 are x = -3, -2, -1, 0 (4 values). Integer values in 5 < x < 10 are x = 6, 7, 8, 9 (4 values). Total = 8.

## Q9
**difficulty:** Hard
**type:** Problem Solving
**topic:** Quadratics

If the roots of the equation x^2 + bx + c = 0 are each increased by 2, the resulting equation is x^2 - 6x + 5 = 0. What is the value of b + c?

- A) -5
- B) -3
- C) 1
- D) 3
- E) 5

**answer:** A
**explanation:** The equation x^2 - 6x + 5 = 0 factors as (x - 1)(x - 5) = 0, giving roots 1 and 5. Since these roots are each 2 more than the original roots, the original roots are 1 - 2 = -1 and 5 - 2 = 3. For x^2 + bx + c = 0 with roots -1 and 3: sum of roots = -b, so -1 + 3 = -b, giving b = -2. Product of roots = c, so (-1)(3) = c, giving c = -3. Therefore b + c = -2 + (-3) = -5.

## Q10
**difficulty:** Hard
**type:** Problem Solving
**topic:** Exponents

If 4^(a+1) = 8^a, what is the value of a?

- A) 1
- B) 2
- C) 3
- D) 4
- E) 6

**answer:** B
**explanation:** Rewrite both sides with base 2: 4^(a+1) = (2^2)^(a+1) = 2^(2a+2). 8^a = (2^3)^a = 2^(3a). Setting exponents equal: 2a + 2 = 3a, so a = 2.

## Q11
**difficulty:** Hard
**type:** Problem Solving
**topic:** Functions

For all positive integers n, let f(n) = n^2 - n. What is the value of f(f(3))?

- A) 6
- B) 12
- C) 30
- D) 42
- E) 56

**answer:** C
**explanation:** First, f(3) = 3^2 - 3 = 9 - 3 = 6. Then f(f(3)) = f(6) = 6^2 - 6 = 36 - 6 = 30. Trap answer D (42) could come from computing f(7) by mistake.

## Q12
**difficulty:** Hard
**type:** Problem Solving
**topic:** Systems of Equations

If xy = 6 and x^2 + y^2 = 20, what is the value of (x + y)^2?

- A) 26
- B) 28
- C) 30
- D) 32
- E) 36

**answer:** D
**explanation:** Using the identity (x + y)^2 = x^2 + 2xy + y^2: (x + y)^2 = (x^2 + y^2) + 2xy = 20 + 2(6) = 20 + 12 = 32. Trap answer A (26) comes from adding 20 + 6 without doubling the xy term.

## Q13
**difficulty:** Easy
**type:** Problem Solving
**topic:** Quadratic Equations

What is the sum of the solutions to x^2 + 4x - 21 = 0?

- A) -7
- B) -4
- C) -3
- D) 3
- E) 7

**answer:** B
**explanation:** By Vieta's formulas, for ax^2 + bx + c = 0, the sum of roots equals -b/a. Here -b/a = -4/1 = -4. You can verify by factoring: (x + 7)(x - 3) = 0 gives roots -7 and 3, which sum to -4. On the GMAT, Vieta's is faster than factoring when you only need the sum or product of roots.

## Q14
**difficulty:** Medium
**type:** Problem Solving
**topic:** Quadratic Equations

If x^2 - 10x + k = 0 has exactly one real solution, what is the value of k?

- A) 5
- B) 10
- C) 20
- D) 25
- E) 50

**answer:** D
**explanation:** A quadratic ax^2 + bx + c = 0 has exactly one real solution when its discriminant b^2 - 4ac = 0. Here (-10)^2 - 4(1)(k) = 0, so 100 - 4k = 0, giving k = 25. The repeated root is x = 5, since x^2 - 10x + 25 = (x - 5)^2.

## Q15
**difficulty:** Medium
**type:** Problem Solving
**topic:** Systems of Equations

If 3x + 2y = 16 and 5x - 2y = 16, what is the value of x + y?

- A) 4
- B) 5
- C) 6
- D) 7
- E) 8

**answer:** C
**explanation:** Adding the two equations eliminates y: 8x = 32, so x = 4. Substitute back into 3x + 2y = 16: 12 + 2y = 16, so 2y = 4, giving y = 2. Therefore x + y = 4 + 2 = 6. The elimination shortcut works whenever the coefficients on one variable are opposites; always verify by plugging back into both original equations.

## Q16
**difficulty:** Hard
**type:** Data Sufficiency
**topic:** Systems of Equations

What is the value of x?

(1) 2x + 3y = 14
(2) 4x + 6y = 28

- A) Statement (1) ALONE is sufficient, but statement (2) alone is not sufficient.
- B) Statement (2) ALONE is sufficient, but statement (1) alone is not sufficient.
- C) BOTH statements TOGETHER are sufficient, but NEITHER statement ALONE is sufficient.
- D) EACH statement ALONE is sufficient.
- E) Statements (1) and (2) TOGETHER are NOT sufficient.

**answer:** E
**explanation:** Statement (1) is one equation in two unknowns — insufficient. Statement (2) is just 2 times Statement (1), so it gives the same information — also insufficient. Together, the two statements are equivalent, not independent, so they still describe a single line with infinitely many (x, y) pairs. The trap is assuming two equations always solve a two-variable system; they must be linearly independent.

## Q17
**difficulty:** Easy
**type:** Problem Solving
**topic:** Absolute Value

If |x - 4| = 7, what is the sum of all possible values of x?

- A) -11
- B) -3
- C) 4
- D) 8
- E) 11

**answer:** D
**explanation:** |x - 4| = 7 splits into two cases: x - 4 = 7 gives x = 11, and x - 4 = -7 gives x = -3. The sum is 11 + (-3) = 8. A useful shortcut: for |x - a| = b, the two solutions are symmetric about a, so their sum is always 2a = 2(4) = 8.

## Q18
**difficulty:** Medium
**type:** Problem Solving
**topic:** Absolute Value Inequalities

For how many integer values of x is |2x + 1| < 9?

- A) 7
- B) 8
- C) 9
- D) 10
- E) 11

**answer:** B
**explanation:** |2x + 1| < 9 means -9 < 2x + 1 < 9. Subtract 1 from each part: -10 < 2x < 8. Divide by 2: -5 < x < 4. The integers strictly between -5 and 4 are -4, -3, -2, -1, 0, 1, 2, 3 — that's 8 integers. Remember: |expr| < k always translates to -k < expr < k, and strict inequalities exclude the endpoints.

## Q19
**difficulty:** Medium
**type:** Data Sufficiency
**topic:** Functions

If f(x) = ax + b where a and b are constants, what is the value of f(5)?

(1) f(1) = 7
(2) f(3) = 13

- A) Statement (1) ALONE is sufficient, but statement (2) alone is not sufficient.
- B) Statement (2) ALONE is sufficient, but statement (1) alone is not sufficient.
- C) BOTH statements TOGETHER are sufficient, but NEITHER statement ALONE is sufficient.
- D) EACH statement ALONE is sufficient.
- E) Statements (1) and (2) TOGETHER are NOT sufficient.

**answer:** C
**explanation:** f(x) = ax + b has two unknowns: a and b. Statement (1) alone gives a + b = 7 — one equation, two unknowns, insufficient. Statement (2) alone gives 3a + b = 13 — also insufficient. Together we have a system: subtracting gives 2a = 6, so a = 3 and b = 4, meaning f(5) = 3(5) + 4 = 19. Two linear equations in two unknowns pin down the function.

## Q20
**difficulty:** Hard
**type:** Problem Solving
**topic:** Algebra Translation

The sum of three consecutive even integers is 18 more than twice the smallest of the three. What is the largest of the three integers?

- A) 10
- B) 12
- C) 14
- D) 16
- E) 18

**answer:** D
**explanation:** Let the three consecutive even integers be n, n + 2, and n + 4. Their sum is 3n + 6. "18 more than twice the smallest" translates to 2n + 18. Setting them equal: 3n + 6 = 2n + 18, so n = 12. The largest integer is n + 4 = 16. The key skill is translating "more than" as addition to the quantity being compared to, not the other way around.
