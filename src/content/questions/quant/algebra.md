## Q1
**difficulty:** Easy
**type:** Problem Solving
**topic:** Linear Equations

If 3x + 7 = 22, what is the value of 6x + 5?

- A) 25
- B) 30
- C) 35
- D) 40
- E) 45

**answer:** C
**explanation:** From 3x + 7 = 22, we get 3x = 15, so x = 5. Then 6x + 5 = 6(5) + 5 = 30 + 5 = 35. Alternatively, 6x + 5 = 2(3x) + 5 = 2(15) + 5 = 35.

## Q2
**difficulty:** Easy
**type:** Problem Solving
**topic:** Exponents

Which of the following is equivalent to (x^3)^4 / x^5?

- A) x^2
- B) x^7
- C) x^12
- D) x^15
- E) x^17

**answer:** B
**explanation:** (x^3)^4 = x^12. Then x^12 / x^5 = x^(12-5) = x^7.

## Q3
**difficulty:** Easy
**type:** Problem Solving
**topic:** Inequalities

If -3x + 9 > 0, which of the following must be true?

- A) x > 3
- B) x > -3
- C) x < -3
- D) x < 3
- E) x = 3

**answer:** D
**explanation:** -3x + 9 > 0 means -3x > -9. Dividing both sides by -3 and flipping the inequality sign gives x < 3.

If x + y = 12 and x - y = 4, what is the value of x?

- A) 4
- B) 6
- C) 8
- D) 10

## Q5
**difficulty:** Medium
**type:** Problem Solving
**topic:** Quadratics

If x^2 - 5x - 14 = 0, what is the product of the two solutions?

- A) -14
- B) -7
- C) 5
- D) 7
- E) 14

**answer:** A
**explanation:** By Vieta's formulas, for x^2 - 5x - 14 = 0, the product of the roots equals c/a = -14/1 = -14. To verify by factoring: x^2 - 5x - 14 = (x - 7)(x + 2) = 0, so x = 7 or x = -2, and 7 x (-2) = -14.

## Q6
**difficulty:** Medium
**type:** Problem Solving
**topic:** Functions

If f(x) = 2x^2 - 3x + 1, what is the value of f(3) - f(1)?

- A) 8
- B) 10
- C) 12
- D) 14
- E) 16

**answer:** B
**explanation:** f(3) = 2(9) - 3(3) + 1 = 18 - 9 + 1 = 10. f(1) = 2(1) - 3(1) + 1 = 2 - 3 + 1 = 0. Therefore f(3) - f(1) = 10 - 0 = 10.

## Q7
**difficulty:** Medium
**type:** Problem Solving
**topic:** Systems of Equations

At a bakery, 2 muffins and 3 scones cost $21, while 4 muffins and 1 scone cost $17. What is the cost of one scone?

- A) $1
- B) $2
- C) $3
- D) $4
- E) $5

**answer:** E
**explanation:** Let m = price of a muffin and s = price of a scone. Equation 1: 2m + 3s = 21. Equation 2: 4m + s = 17. From Equation 2: s = 17 - 4m. Substitute into Equation 1: 2m + 3(17 - 4m) = 21, so 2m + 51 - 12m = 21, giving -10m = -30, so m = 3. Then s = 17 - 4(3) = 17 - 12 = 5. One scone costs $5.

## Q8
**difficulty:** Medium
**type:** Problem Solving
**topic:** Inequalities

If 2 < |x - 3| < 7, how many integer values of x satisfy the inequality?

- A) 6
- B) 7
- C) 8
- D) 9
- E) 10

**answer:** C
**explanation:** |x - 3| > 2 means x > 5 or x < 1. |x - 3| < 7 means -4 < x < 10. Combining: -4 < x < 1 or 5 < x < 10. Integer values in -4 < x < 1 are x = -3, -2, -1, 0 (4 values). Integer values in 5 < x < 10 are x = 6, 7, 8, 9 (4 values). Total = 8.

## Q9
**difficulty:** Hard
**type:** Problem Solving
**topic:** Quadratics

If the roots of the equation x^2 + bx + c = 0 are each increased by 2, the resulting equation is x^2 - 6x + 5 = 0. What is the value of b + c?

- A) -5
- B) -3
- C) 1
- D) 3
- E) 5

**answer:** A
**explanation:** The equation x^2 - 6x + 5 = 0 factors as (x - 1)(x - 5) = 0, giving roots 1 and 5. Since these roots are each 2 more than the original roots, the original roots are 1 - 2 = -1 and 5 - 2 = 3. For x^2 + bx + c = 0 with roots -1 and 3: sum of roots = -b, so -1 + 3 = -b, giving b = -2. Product of roots = c, so (-1)(3) = c, giving c = -3. Therefore b + c = -2 + (-3) = -5.

## Q10
**difficulty:** Hard
**type:** Problem Solving
**topic:** Exponents

If 4^(a+1) = 8^a, what is the value of a?

- A) 1
- B) 2
- C) 3
- D) 4
- E) 6

**answer:** B
**explanation:** Rewrite both sides with base 2: 4^(a+1) = (2^2)^(a+1) = 2^(2a+2). 8^a = (2^3)^a = 2^(3a). Setting exponents equal: 2a + 2 = 3a, so a = 2.

## Q11
**difficulty:** Hard
**type:** Problem Solving
**topic:** Functions

For all positive integers n, let f(n) = n^2 - n. What is the value of f(f(3))?

- A) 6
- B) 12
- C) 30
- D) 42
- E) 56

**answer:** C
**explanation:** First, f(3) = 3^2 - 3 = 9 - 3 = 6. Then f(f(3)) = f(6) = 6^2 - 6 = 36 - 6 = 30. Trap answer D (42) could come from computing f(7) by mistake.

## Q12
**difficulty:** Hard
**type:** Problem Solving
**topic:** Systems of Equations

If xy = 6 and x^2 + y^2 = 20, what is the value of (x + y)^2?

- A) 26
- B) 28
- C) 30
- D) 32
- E) 36

**answer:** D
**explanation:** Using the identity (x + y)^2 = x^2 + 2xy + y^2: (x + y)^2 = (x^2 + y^2) + 2xy = 20 + 2(6) = 20 + 12 = 32. Trap answer A (26) comes from adding 20 + 6 without doubling the xy term.
